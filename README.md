# SNS MultiPost v2.1 🚀

複数のSNS（Instagram, YouTube, LINE, X, Facebook, TikTok）への投稿準備・管理・分析を効率化するWebアプリケーションです。

## 🌟 主な機能
- **明るいポップデザイン**: 2026年スタイルの洗練されたUI。
- **メディアライブラリ**: 画像・動画のストック管理。Google Driveのリンクも直接登録可能。
- **クイック投稿**: テンプレートとメディアを選んで、即座に投稿準備（コピー＆SNS直行）。
- **投稿スケジュール管理**: カレンダー形式で予定を視覚化。
- **分析データ管理**: 再生数、いいね、コメントを手動記録。CSV出力でスプレッドシート分析が可能。
- **完全ローカル実行**: データはブラウザの `localStorage` に保存されるため、プライバシーも安心。

## 🚀 使い方
1. `index.html` をブラウザで開きます。
2. 左メニューから各機能にアクセスしてください。

## 🛠 技術構成
- HTML / CSS / JavaScript (Vanilla)
- 外部APIを使用せず、安全に投稿準備が可能。
